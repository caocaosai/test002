สวัสดีครับ วันนี้ Zinglecode ขอเปิดตัวคอร์ส Workshop HTML, CSS (Bulma), Javascript สไตล์เร้าใจจัดๆ เนื้อหาจะเป็นการเชื่อมต่อทั้ง 3 ภาษานี้เข้าด้วยกัน เพื่อเริ่มต้นสร้างเป็นผลงานหน้าเว็บเพจสวยๆ และ UI เด็ดๆขึ้นมา

ราคาโปรโมชั่นเปิดตัว ลดจากราคาเต็ม 3,500 บาท เหลือเพียง 1,800 บาท
